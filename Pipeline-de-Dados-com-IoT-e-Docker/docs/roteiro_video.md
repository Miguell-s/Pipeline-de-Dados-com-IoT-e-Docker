## [0:30 - 1:15] Objetivo e Tecnologias Utilizadas

**[Visual: Lista de tecnologias aparecendo uma a uma na tela]**

**Narrador**:  
"Nosso objetivo é criar um pipeline completo que:
1. Receba dados de sensores IoT de temperatura
2. Processe e limpe esses dados
3. Armazene-os em um banco de dados
4. Visualize tudo em um dashboard interativo

Para isso, utilizamos tecnologias modernas e profissionais:
- **Python** para processamento de dados
- **Docker** para containerização
- **PostgreSQL** como banco de dados
- **SQLAlchemy** como ORM
- **Pandas** para manipulação de dados
- **Streamlit** e **Plotly** para o dashboard
- **Git e GitHub** para controle de versão"

---

## [1:15 - 2:00] Funcionamento do Pipeline

**[Visual: Diagrama animado do pipeline - da esquerda para a direita]**

**Narrador**:  
"Vamos entender como funciona o nosso pipeline, passo a passo.

Primeiro, temos a **ingestão de dados**: coletamos leituras de temperatura de dispositivos IoT instalados em ambientes internos e externos.

Em seguida, o **processamento**: usamos Python e Pandas para limpar os dados, tratar valores ausentes e converter formatos de data.

Depois, o **armazenamento**: inserimos todos os dados em um banco PostgreSQL, que está rodando em um container Docker para garantir portabilidade.

E para finalizar, criamos **views SQL** otimizadas que facilitam nossas análises!"

## [2:00 - 3:15] Demonstração do Dashboard

**[Visual: Gravação da tela do dashboard em funcionamento]**

**Narrador**:  
"Agora, a parte mais interessante: o nosso **dashboard interativo** desenvolvido com Streamlit!

Aqui na parte superior, temos as métricas principais: total de dispositivos, total de leituras e temperatura média geral.

Primeiro gráfico: **média de temperatura por dispositivo**, em um gráfico de barras colorido. Podemos ver claramente quais ambientes estão mais quentes ou mais frios.

Segundo gráfico: **leituras por hora do dia**, em um gráfico de linhas. Isso nos mostra os horários de pico de atividade dos sensores.

E o terceiro gráfico, o meu preferido: **temperaturas máximas e mínimas por dia**, completamente interativo! Podemos passar o mouse para ver os valores exatos e explorar as tendências ao longo do tempo."

## Dicas de Gravação

1. **Preparação**: Teste todo o fluxo antes de gravar
2. **Áudio**: Use um microfone decente para garantir qualidade de som
3. **Visualizações**:
   - Grave a tela do dashboard em alta resolução
   - Use transições suaves entre as seções
   - Adicione texto em tela para destacar pontos importantes
4. **Edição**:
   - Mantenha o ritmo ágil, sem pausas longas
   - Adicione música de fundo leve (sem vocais)
   - Inclua legendas para acessibilidade
